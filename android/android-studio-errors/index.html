This is a warning saying that the ADB binary found is obsolete. It is not a critical error.

Action to take

Check the ADB version.

The ADB's location is found in the alert box..

Navigate to the platform-tools directory where adb.exe is located.

Execute the following command in the Command Prompt.

adb.exe --version

Possible Solution

  • Uninstall older SDK build tools versions, leaving only the most recent one
