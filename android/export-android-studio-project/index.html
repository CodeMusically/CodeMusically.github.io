It is very useful to export an Android Studio project as a zip file. An Android Studio project contains thousands of files and can easily take up a lot of space. By exporting the project as a zip file, only the essential files are included. This also means that the file size is much smaller than the original one.

Step 1: Open your Android Studio project

Make sure that there is no error in your project.

Step 2: Run Export to Zip File...

From the top menu, select File â†’ Export to Zip File...

Step 3: Run Export to Zip File...

Specify the location and file name for the zip file. Click OK.

The file will be generated in the specified directory.

The original project (13.8MB) contains 439 files in 187 folders.

Now, the zip file is only 139 KB!
